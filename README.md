### User Story

```
xx스토어의
제품 판매율을 늘리기 위해,
학생을 대상으로 제품들을 할인하는 기획을 내놓았다.
```

### 소비자

- xx스토어에 회원 가입을 할 수 있어야 한다.
- 로그인 및 로그아웃을 할 수 있어야 한다.
- 학생 인증을 통해 할인된 가격으로 제품을 구매할 수 있어야 한다.
- 전체 주문 내역부터, 월별 주문 내역까지 확인 가능해야 한다.
- 구매를 취소할 수 있어야한다.
- 회원 가입했던 정보를 수정할 수 있어야 한다.

### 관리자

- xx스토어에 제품을 등록, 수정, 삭제할 수 있어야 한다.
- 로그인 및 로그아웃을 할 수 있어야 한다.
